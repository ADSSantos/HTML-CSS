Para fazer o efeito zebrado usamos um pseudo elemento,
tbody > tr:nth-child(2n) { background-color: lightgray; } tr é filho do tbody que recebe o pseudo nth-child()
o valor usado neste parametro define quantas linhas será afetada com a cor desejada,tambem aceita valores pares e impares em ingles.
